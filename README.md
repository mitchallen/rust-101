## References

* [Getting Started with Rust (Hello World, Mac)](https://scriptable.com/rust/getting-started-with-rust-hello-world)

## Usage

### Make

```sh
make all
```

### Build

```sh
cargo build
```

### Run

```sh
cargo run
```

Run with less verbosity:

```sh
cargo run --quiet
```

### Test

```sh
cargo test
```
